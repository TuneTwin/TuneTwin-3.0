# TuneTwin

## Tech Stack
- Backend: Django + Django REST Framework
- Database: PostgreSQL
- Frontend: React

## Structure

- `tunetwin/`
  - `backend/`
  - `frontend/`
  - `docs/`

## Branching Model
- `main`: stable
- `dev`: integration
- `feature/*`: work branches

## To Set Up

- Create a virtual environment with `python3 -m venv ttvenv`
- Run the virtual environment with `source ttvenv/bin/activate`
- Install Django inside the virtual environment: `python3 -m pip install Django`
- Type `deactivate` when you're done
